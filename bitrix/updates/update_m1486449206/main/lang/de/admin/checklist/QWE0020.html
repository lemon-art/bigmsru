Die Plattform verfügt über ein integriertes Paket der Autotests, mit denen korrekte Funktionsweise einzelner Bestandteile überprüft wird. Alle Tests müssen erfolgreich gemacht werden, um eine sichere Funktionsweise des Webprojektes sicherstellen zu können.

Im Bereich "Einstellungen > Tools > Systemprüfung" auf dem Tab "Intranet-Funktionsfähigkeit" müssen alle Tests gemacht werden.
